# Dotfiles

Personal Linux dotfiles for Fedora + Hyprland.

## Components

- Hyprland
- Waybar
- Wofi
- Mako
- Kitty
- Zsh

## Installation

```bash
git clone https://github.com/Qu0i/dotfiles.git
cd dotfiles
./install.sh
```

---

> [!WARNING]
> These dotfiles are tailored to my personal setup and may require adjustments for other systems.
